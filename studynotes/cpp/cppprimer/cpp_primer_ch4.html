An expression is composed of one or more operands and yields a result when it is evaluated. The simplest form of an expression is a single literal or variable.

In terms of overloaded operators, the number of operands and the precedence and the associativity of the operator cannot be changed.

Fundamentals

Lvalues and Ravlues

Every expression in C++ is either an rvalue or an lvalue. In C++, an lvalue expression yields an object or a function. However, some lvalues, such as const objects, may not be the left-hand operand of an assignment. Moreover, some expressions yield objects but return them as rvalues. Roughly speaking, when we use an object as an rvalue, we use the object’s value (its content). When we use an object as an lvalue, we use the object’s identity (its location in memory).
